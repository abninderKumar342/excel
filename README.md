# Heating Calculation Excel Project

## 📌 Overview
This project is an Excel-based solution for performing heating-related calculations. It helps in organizing and analyzing heating data efficiently.

## 📂 Files Included
- Heating PL.xlsx → Main Excel file containing calculations and data

## ⚙️ Features
- Structured data format
- Easy-to-use Excel sheet
- Useful for basic heating analysis

## 🎯 Purpose
This project was created to demonstrate Excel skills such as:
- Data organization
- Basic calculations
- Practical problem-solving

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Modify values as per your requirement
